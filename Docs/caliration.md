# 机械臂关节标定记录

> 修改时间: 20260527

## joint0 / ID0 / 底座

- home: 1500
- min: 550
- max: 2450
- 观察视角：从上往下看
- pos 增大：逆时针
- pos 减小：顺时针
- 备注：
  - 500 基本可达，旋转约175度
  - 2500 基本可达，旋转约180度

## joint1 / ID1 / 大臂

- home: 1500
- min: 700
- max: 2200
- 观察视角：侧视，板子在左侧
- pos 增大：逆时针，向前倒 / 向前伸
- pos 减小：顺时针，向后倒 / 向后收
- 软件方向：暂定 pos 增大为 joint1 正方向
- 备注：
  - 700 向后倒（竖直夹角约70度），再向后大臂会碰到底座
  - 2200 向前倒（竖直夹角约80度），再向前大臂会碰到数据线
  - 安全限幅可先使用 750~2150，后面根据走线情况再放宽

## joint2 / ID2 / 小臂

- home: 1500
- min: 900
- max: 2250
- 观察视角：侧视，板子在左侧
- pos 增大：顺时针，向后倒 / 向后伸
- pos 减小：逆时针，向前倒 / 向前伸
- 软件方向：暂定 pos 减小为 joint2 正方向
- 备注：
  - 850 向前倒（竖直夹角约90度），再向前舵机会支撑不住重力，抖
  - 2300 向后倒（竖直夹角约70度），再向后舵机会支撑不住重力，抖
  - 安全限幅使用 900~2250，后面根据走线情况再放宽

## joint3 / ID3 / 腕部折叠

- home: 1500
- min: 800
- max: 2150
- 观察视角：侧视，板子在左侧
- pos 增大：逆时针，向前倒 / 向前伸
- pos 减小：顺时针，向后倒 / 向后收
- 软件方向：暂定 pos 增大为 joint3 正方向
- 备注：
  - 750 向后倒（竖直夹角约80度），姿态不好看了
  - 2200 向前倒（竖直夹角约85度），姿态已经很难看了，且舵机和小臂可能有干涉
  - 安全限幅先使用 800~2150，后面根据走线情况再放宽

## joint5 / ID5 / 夹爪

- home: 1500
- min: 1100
- max: 1700
- pos 增大：夹爪闭合
- pos 减小：夹爪张开
- 备注：
  - 1000 可达，但到1100连杆已经基本水平，夹爪无法再张大
  - 1650 时夹爪间距约 5~7mm
  - 1690 时夹爪间距约 1~2mm，未顶死，作为当前安全闭合上限
  - 1700 时，夹爪间距大概还有0-1mm，多次实验有时剩1mm，有时贴合，但胶套没有顶死，还有0.5mm左右的空间
